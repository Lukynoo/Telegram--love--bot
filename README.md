# 🤖 Telegram Love Bot
Jednoduchý Python skript pro automatické odesílání náhodných zpráv přes Telegram API.

## ✨ Co projekt řeší?
Tento bot automatizuje každodenní rutinu odesílání povzbuzujících zpráv. Využívá knihovnu `python-telegram-bot` a plánovač `schedule`.

## 🛠️ Použité technologie
* **Python 3.x** - Jádro aplikace.
* **Asyncio** - Asynchronní odesílání zpráv (efektivnější komunikace s API).
* **Schedule** - Lehké a čitelné plánování úloh v čase.

## 🔐 Bezpečnost
Kód je připraven pro nasazení s využitím proměnných prostředí, aby citlivé údaje (API Tokeny) zůstaly v bezpečí a nebyly součástí veřejného kódu.
